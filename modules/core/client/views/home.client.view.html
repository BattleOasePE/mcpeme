The old version of mcpe.me has been decommissioned and replaced with this website. You will need to create an account and sign in. Then you can click "Records" and "Create records" in the menu bar. Enter your details and your record will be created instantly. On [date], all old records were removed from our DNS provider. We have dumped record data publicly. Now any user can register any name they like. However, until [date], all old record holders can claim records which they used to own and have been taken by another user by messaging @falkirknh on Twitter. When assessing claims for old records, the old service policy will be used. Verification is a process which ensures that the record owner is also the server owner. Only MCPE servers can be verified. You may be requested to verify your server by our staff for security reasons.

You can verify a server by setting the title of the server to the "Record ID" and restarting the server. mcpe.me will verify you server a few seconds after you add the server. If your server is confirmed to be an MCPE server (but doesn't have the verification code set), mcpe.me will try again in 20 minutes. You can force mcpe.me to re-verify by submitting the edit form without any changes. Verification is not instant, it will take about 15 second for mcpe.me to complete a verification. If you don't want to see ads, consider making a donation and then ads will be hidden from your account. The new service generates and uses an array of tokens and key to operate.
  • User ID - this is a unique identifier for a user that can't be changed. It is useful for reporting users.
